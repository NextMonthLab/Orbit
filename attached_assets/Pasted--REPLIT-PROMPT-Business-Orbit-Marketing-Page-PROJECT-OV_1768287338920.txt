# REPLIT PROMPT: Business Orbit Marketing Page

## PROJECT OVERVIEW

Build a high-converting, product-led SaaS marketing page for Business Orbit ‚Äî a conversational business intelligence platform that positions as a new software category. The page must establish category definition (not feature comparison), optimize for self-serve signup, and communicate premium positioning for established companies.

## TECHNICAL REQUIREMENTS

### Stack

- **Framework**: Next.js 14+ with App Router
- **Styling**: Tailwind CSS
- **Components**: React with TypeScript
- **Animations**: Framer Motion for scroll animations and micro-interactions
- **Forms**: React Hook Form for email capture
- **Deployment**: Vercel-ready configuration

## BRAND GUIDELINES REFERENCE

### Logo Usage

- Minimum size: 120px width
- Clear space: Equal to height of ‚ÄúO‚Äù in ‚ÄúOrbit‚Äù
- Always on dark background (#1a1b2e) or transparent
- Never distort aspect ratio
- Never add effects (drop shadow, outer glow, etc.)

### Gradient Application

- Primary use: Accents, borders, text highlights, CTAs
- Never: Full background gradient (too overwhelming)
- Opacity: 80-100% for primary elements, 20-40% for subtle effects
- Direction: 135deg (diagonal) maintains brand consistency

### Iconography Style

- Outlined style preferred (not filled)
- Gradient stroke treatment for key icons
- Consistent stroke weight: 2px
- Icons from same family (Lucide, Heroicons, or Phosphor)
