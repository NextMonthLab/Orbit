Notes you should give RAPID (so it behaves consistently)
	•	IDs are mandatory and stable. Never regenerate IDs on re-run.
	•	Do not invent RSS. If unsure, set rss_url: null and use page_monitor.
	•	Specs must have source_url. If unknown, store “Unknown” and keep the source URL.
	•	Assets are references, not downloads. Keep source_url and licensing notes.
	•	Seed packs should be “good enough”, not exhaustive. The Pulse loop will grow it.
